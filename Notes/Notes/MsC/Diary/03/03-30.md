When does 'processed' directory in data get created? And what is it for?
Processed directory contains cell graphs of both test and train data.
When GraphEmbed is called, it uses the graphs to make sc predictions. The graphs are made when GeoMX method process is called

The GeoMX dataset inherits from torch.geometric.data, and in that parent class, it calls function process whenever the child is initialized, meaning that when GeoMX is called, it automatically runs process.

So, the dataset is initialized, scans the folder depending on argument split, and creates cell graphs if missing.

So to make predictions, i just need to use cellprediction, argument embed_gnn_data, and assign which model to use and the correct parameters for that model.
