Individual conditional expectation. Equivalent of PDP but for individual conditional expectation.

Visualizes the dependence of prediction on a feature for each instance separately, one line per instance.

![[Pasted image 20260112174559.png]]

Compared to one line overall in PDP, which is an average of the lines in an ICE plot. Each line is one datapoint, and you slide the feature to be investigated.

* Intuitive
* ICE curves can only display one feature meaningfully
* Plot can become overcrowded
