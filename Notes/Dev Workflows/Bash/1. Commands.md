| **Commands**      | Output                                                                                                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ls -a             | List hidden files                                                                                                                                                                        |
| ln -s /path /path | Create a link to a directory. WARNING! When removing a link, don't use `rm path/` as this will delete contents instead! Make sure it doesn't end with `/`, so the right way is `rm path` |
| ls -lh            | List contents in a human-readable format                                                                                                                                                 |
| `ls *abc*`        | List with substring                                                                                                                                                                      |
| `du -sh *`        | Memory size of current directory contents                                                                                                                                                |
| `ls *pattern`     | List only of files with pattern                                                                                                                                                          |
| `ls -1 \| wc -l`  | List how many files in current directory                                                                                                                                                 |
