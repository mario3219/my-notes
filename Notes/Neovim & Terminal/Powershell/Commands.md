**Data handling**

| Commands                | Descriptions            |
| ----------------------- | ----------------------- |
| cd                      | Print current directory |
| mkdir foldername        | Create directory        |
| rmdir /s /q foldername  | Remove directory        |
| type nul > file.txt     | Create empty text file  |
| move source destination | Move                    |
| ren oldname newname     | Rename                  |
| copy source destination | Copy                    |
