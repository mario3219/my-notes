
Create
```
git branch <name>
```

See local branches
```
git branch
```

See remote branches
```
git branch -r
```

Navigate to branch
```
git checkout <name>
```

Delete local branch
```
git branch --delete <name>
```

Delete remote branch
```
git push origin -d <name>
```

Rename branch
```
git branch -m <new name>
```
